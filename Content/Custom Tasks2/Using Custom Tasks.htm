Working With Custom Tasks

 

Custom tasks can be used in several ways. While some Custom Tasks can be only be used in a Form or a Process Timeline, most Custom Tasks can be used in both. This section primarily discusses the different use cases in which Custom Tasks can be implemented. Custom Tasks must also imported into your system periodically, as they are updated by BP Logix, which is also discussed in this section of the documentation.
