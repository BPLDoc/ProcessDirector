Reconstructing the Sample project

Now that we’ve seen the sample project in action, and covered some of the basics of building a Form and process, we’re going to go deep into the details of how the sample project works by recreating it from scratch. We will build each piece, step by step. At every point along the way, this guide will explain how and why we are taking each action.

As you reconstruct the sample project, be sure to use Internet Explorer as your web browser. For regular Process Director users, i.e., those who do not create Process Director objects or design processes, any browser will work properly. But, certain features of Process Director do not work with other web browsers during the design process. Process Director uses Microsoft's ActiveX technology in the design process, so only Internet Explorer will fully support this technology.

The Sample Project for this guide contains examples of a Forms using both the Word Form Builder and the Online Form Designer. Though this guide refers to the process used to create the Form using the Online Form Designer, the Form controls and design are essentially the same using either form creation tool. We recommend the use of the Online Form Designer in nearly every use case.
