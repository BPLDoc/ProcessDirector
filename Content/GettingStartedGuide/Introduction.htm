Introduction

This guide is designed to help you get started quickly with Process Director. In the next few sections, we'll cover the most basic fundamentals of the product and how to build a simple application.

First, we'll cover the basic design concepts behind the product, to give you an idea of the objects and terminology you'll be dealing with. From there, we'll move to the form definition, and give you a brief outline of how form controls and form field properties relate to each other, discuss the concept of event mapping for form fields, and briefly discuss the Dropdown Object for use with dropdown form fields.

After that, we'll provide an outline of the steps needed to create a Process Director application, giving you links to the specific section of the documentation that address the configuration options available for each step.

While we hope this documentation will help you get started with the product, please know that it is not a substitute for the Process Director training we offerto all customers. For training on how to use the product, please contact your sales representative for a login code to our Online Training Portal. The training portal contains several self-paced training courses that cover all aspects of using Process Director to create low-code/no-code applications.
