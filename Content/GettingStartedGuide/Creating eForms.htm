Creating Forms

Process Director provides three methods for creating Forms: The Online Form Designer, the Microsoft Word Form Builder, and creating custom form controls using Visual Studio.

Online Form Designer

For users of Process Director v4.06 and higher, an Online Form Designer is built into the Process Director Form definition. This is the primary method you should use for creating forms. The Online Form Designer enables you to build Forms in the Browser, and requires no additional software or plugins to use. Forms templates can be built using any modern browser. For detailed information on how to use this method to build a Form template, please see the documentation for the Online Form Designer in the Implementer's Reference Guide, or the remainder of this document.

For the Getting Started Guide, we will assume that you are using the Online Form Designer.

Visual Studio

Developers can, when necessary, use a Custom ASCX control to create Forms. You can choose the ASCX format when creating the Form definition's type. Once the Form Definition has been crated, you can open the ASCX Form in Visual Studio.

Using Visual Studio requires that you download and install the Visual Studio plugin that contains the toolbox controls for Process Director. For detailed information about this plugin, and how to install and use it with Visual Studio, please see the Installing the BP Logix Visual Studio Plug-in and Creating ASP.NET Forms topics in the Developer's Reference Guide.
