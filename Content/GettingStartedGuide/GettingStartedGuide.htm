Getting Started Guide Introduction

The Getting Started Guide is not a reference document. Instead it is designed as a brief, self-service training course to explain some of the basic concepts of Process Director, and to recreate a small project implementation. In this document, the following subjects will be covered:

  • Review an existing sample project and submit a Form to start a simple approval process.
  • A step-by-step guide to reconstructing the sample project, which includes key conceptual explanations.
  • A brief overview of how to analyze running processes.
  • An introduction to Knowledge Views and step-by-step instruction in creating one.

This introductory course is mainly designed for users who need the minimal knowledge necessary to work effectively with process Director during a Trial Period.
