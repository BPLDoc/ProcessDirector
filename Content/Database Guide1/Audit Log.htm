Audit Log

The audit log contains data recording most events in an installation of Process Director. The audit log displays records in the following CSV format:

Date, Event, User ID, IP, Text, Object Name, Object Type, UID, OID, PID, Impersonate User ID, Impersonate UID

Where the columns are defined as follows:

Date: the date and time the event happened.

Event: the type of event. The event type can be any of the following:

  • Login
  • Logoff
  • AuthenticateFailed
  • DeleteObject
  • CreateObject
  • UpdateObject
  • MoveObject
  • PasswordChange
  • CreateUser
  • DeleteUser
  • CreateGroup
  • DeleteGroup
  • WebService
  • TaskCompleted
  • TaskReassigned
  • TaskRemoved
  • Signature
  • RollbackActivity
  • CancelActivity
  • RestartActivity
  • Restart Timeline
  • JumpToStep
  • CancelStep
  • StopStep
  • StopWorkflow
  • RestartWorkflow
  • UpdateUser
  • AddUserToGroup
  • RemoveUserFromGroup
  • RemoveObjectFromParent
  • DownloadDocument
  • ViewFormData
  • PermissionAdded
  • PermissionRemoved
  • PermissionUpdated
  • PermissionReplicated
  • PermissionGiven
  • PermissionDeniedView
  • PermissionDeniedModify
  • PermissionDeniedDelete
  • PermissionDeniedExecute
  • PermissionDeniedAdmin
  • ImpersonateOn
  • ImpersonateOff
  • DelegateOn
  • DelegateOff
  • UserDisabled
  • UserEnabled
  • AdminReplaceUser
  • AdminPageAccess
  • ImportObjects
  • ExportObjects
  • CustomApp
  • FormDataChange
  • ViewForm
  • AdminPageUpdate
  • UpdateDocumentProps
  • RollbackDocument
  • CancelDocumentCheckout
  • CheckoutDocument
  • AnonymousAccess
  • ObjectLock
  • ObjectUnlock
  • ObjectUnlockOther
  • ViewCaseData
  • CaseDataChange
  • CaseModeEntered
  • ReInit
  • UserLockedOut
  • AddGroupToPartition
  • RemoveGroupFromPartition
  • AddUserToPartition
  • RemoveUserFromPartition
  • AddAppEvent
  • UpdateAppEvent
  • DeleteAppEvent

User ID: the ID of the user that caused the event.

IP: the IP address the user was logged on with.

Text: an explanation of the event

Object Name: the name of the object involved in the event

Object Type: the type of the object involved in the event

UID:  the UID of the user affected by the event.

OID: the ID of the object affected by the event.

PID: the ID of the process involved in the event.

Impersonate User ID: the ID of the user that was impersonating the user.

Impersonate UID: the UID of the user that was impersonating the user.
