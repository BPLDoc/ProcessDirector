This document lists the significant recent changes or updates to the documentation, along with the change dates. This is not a comprehensive or detailed log of every single change to every topic. It records the general types of change that were made for each revision of the documentation. For instance, it does not describe every single typographical error fixed on every page, but instead might simply note the changes as "typo fixes" or some similar verbiage. Minor documentation changes (such as minor typo fixes), or interim changes made in the course of larger revision projects, may not be included below.
