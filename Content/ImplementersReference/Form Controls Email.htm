Email Template Controls (Word Form Builder)

Process steps generally notify users using a pre-built email template. These templates use different Form controls to contain the email’s data and provide links to the necessary locations.

EmailData

The EmailData control sets the email’s subject, “from” address and name, and priority.

You can set all these properties in the control’s properties window. You will generally use system variables to dynamically set the email data based on who and what task has triggered the email.

When you select the Add process Attachments or Add Form Attachments properties, you must supply a Group Name. Only attachments specified by their Group Name will be attached to the email. As a general practice, you should always apply Group Names for attachments.
