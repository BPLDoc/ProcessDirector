BP Logix Product Documentation

Click here to see how the Process Director documentation system works.

To get started quickly, you can go directly to the Quick Links Page or use the Quick Links menu at the top of the page to see a collection of links to the most often-used features of Process Director.

A new feature, CustomGPT, has been added to the documentation. You can access the CustomGPT chatbot via the BP Logix icon in the bottom right corner of the screen and ask it any information you'd like about Process Director. You can learn more about this new feature in the Documentation Guide's AI Chat topic.

The documentation for Process Director is largely reference documentation. For training on how to use the product, please contact your sales representative for a login code to our Online Training Portal. The training portal contains several self-paced training courses that cover all aspects of using Process Director to create low-code/no-code applications. A link to the training portal is also available in the main menu at the top of each page.

This documentation references the most current version of Process Director. Documentation for versions of Process Director v3.82 and below, please refer to the legacy documentation found at archive.bplogix.com.

Live Product Webinars

BP Logix offers two recurring webinars.

The Training Webinar is held every Thursday from 10AM Pacific (1PM Eastern) until 12PM Pacific (3PM Eastern). Each week, a specific process Director topic will be taught live during the webinar. You can see a schedule of Training Webinar topics at the BP Logix Support site. Click here to register for the Training Webinar.

The Feature Spotlight Webinar is held on the second Tuesday of every month, and runs from 10AM Pacific (1PM Eastern) until 10:30AM Pacific (1:30PM Eastern). Each month, the webinar will provide updates on new or upcoming features in Process Director. Click here to register for the Feature Spotlight Webinar.

Registering for the webinars will enable you to attend, and ensure that you receive notification reminders prior to each webinar.
