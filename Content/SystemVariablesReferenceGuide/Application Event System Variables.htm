Application Event System Variables

App Events

Returns

This system variable returns information about a Milestone. These events are also displayable in a Journal control.

SysVar Tag

{APP_EVENTS, occurrence=N, eventname="String", format="user"|"message"|"time"|"date"|"datetime"}

Modifiers

occurrence: The number that identifies the iteration number of the events in an iterating loop.

eventname: The name of a specific event to return.

format: The desired format of the data to return about the event.

Alternate Syntax

{JOURNAL_ITEMS}
