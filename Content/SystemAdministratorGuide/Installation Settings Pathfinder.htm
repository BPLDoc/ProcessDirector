Pathfinder Variables

This topic discusses a feature that is under active development, and is subject to change without notice.

For Process Director v6.1.500 and higher, administrators of installations using the Subscription license will have access to the Pathfinder Variables page. Pathfinder is the name BP Logix has given to an online help system that uses Microsoft's Azure AI Foundry to provide an online help system within the product, based on the published product documentation. This feature was created for customers using the platform bases Approvia and PubPro products, which are approval platforms designed as Process Director-based custom applications.

In most cases, the settings on this page will be configured by BP Logix personnel, and only for Cloud-based installations, using the BP Logix Azure platform. Otherwise, the settings on this page will be blank. BP Logix will not provide any additional guidance, beyond the brief explanations given on this page, for using or configuring the Azure AI Foundry for Pathfinder.

When implemented, a Pathfinder icon will appear in the user interface of the product, as described in the User Interface topic of the Implementer's Guide. This icon will open the UI for this feature, which appears as a chatbot window in which users can ask the AI agent for information about, or help with, using Process Director (or the PubPro or Approvia products).

The Pathfinder Variables page contains the properties listed below.

Model Endpoint URL

This is the fully-qualified URL of the deployed model endpoint in Azure AI Foundry, which will be used to send inference requests.

 Model Deployment Name

The Name of the model deployment that was created Azure AI Foundry to host the AI model that was trained on the relevant product documentation.

 Model API Key

API key used to authenticate requests to the model endpoint. This field is obscured for security reasons.

 AI Search Endpoint

The base URL of the Azure AI Search service used for indexing and querying data.

 AI Search Index Name

Name of the Azure AI Search index used for semantic queries.

 AI Search Query Key

Query key used to authenticate read-only access to the Azure AI Search index. This field is obscured for security reasons.

 AI Search Semantic Config

The name of the semantic configuration within the Azure AI Search index, used to enable semantic ranking and improve search relevance.

 AI Search Embedding Deployment Name

The name of the deployed embedding model in Azure OpenAI used for vector search operations.

 Storage Account Name

Name of the Azure Storage Account used to store files or documents accessed by the AI system.

 Storage Tenant ID

Azure Active Directory Tenant ID associated with the storage account. Required for authentication.

 Storage Client ID

Client ID (Application ID) of the Azure AD app registration used to access the storage account.

 Storage Client Secret

Client secret for the Azure AD app registration. Used for secure authentication to storage. This field is obscured for security reasons.

 Storage Container Name

Name of the blob container within the Azure Storage Account where files or documents are stored.

 Azure AI Foundry Advanced Options (optional)

a JSON-formatted string containing the advanced or additional options to make Pathfinder work.
