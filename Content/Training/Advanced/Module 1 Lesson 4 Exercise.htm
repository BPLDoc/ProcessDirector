Skill Exercise: Adding Knowledge Views to a Form

Exercise Performance Tasks

Instructions

Complete the following tasks:

  1. Create a new Form, and check it out for editing in Microsoft Word.
  2. Add a Section to the Form that contains an input control named "SequenceNumber", and, if desired, a few other controls of your choice.
  3. Add a second section to the Form. In the second section, add a button control, then add two Knowledge View controls.
  4. In the first section, configure the input control to use a Sequence Number as the default value
  5. Using the Q&A project provided with the Sample Project, configure the button and KView controls to implement the Q&A process for the Form, using the SequenceNumber as the query string for the Knowledge Views.
  6. Test the Q&A process to ensure it works with your Form.
