Skill Exercise: Visibility and Enabling

Exercise Performance Tasks

Instructions

Complete the following tasks:

  1. Create a new Form and open the Form for editing.
  2. Add two (2) Section controls to the Form. In each Section control, place some controls.
  3. Save and Check In the Form.
  4. Run the Form to see that both sections are visible.
  5. In the Form definition, set one of the Section controls to be hidden from the Timeline Initator.
  6. Run the Form to see that the configured section is no longer visible to you.
