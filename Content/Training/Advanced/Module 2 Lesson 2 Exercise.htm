Skill Exercise: Iteration

Exercise Performance Tasks

Instructions

Complete the following tasks:

  1. Create a new Process Timeline.
  2. Create a Parent activity as the first Process Timeline activity.
  3. Create three User activities as children of the Parent activity, and name the activities:
    1. Supervisor Approval
    2. Manager Approval
    3. Director Approval
  4. Create the appropriate dependencies so that the three user activities constitute a three-step process.
  5. Assign test users to the User activities, and create an "Approve", "Deny" and "Resubmit" result for each.
  6. Set the iterating loop to:
    1. Restart when any User activity result is "Resubmit".
    2. Exit the loop when any user result is "Deny".
  7. Test the iterating loop to ensure it works correctly.
  8. Save the Process Timeline for use in the next lesson.
