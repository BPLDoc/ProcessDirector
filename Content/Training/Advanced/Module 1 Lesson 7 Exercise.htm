Skill Exercise: Configuring Goals

Exercise Performance Tasks

Instructions

  1. Complete the following tasks:
  2. Create a new Goal definition named "Part of Month".
  3. Set the Group in Value Picker property as "Training", and ensure the Goal is active.
    1. Create the following Actions:
      1. Action 1
        1. Set the condition for the first action as "{CURR_DATE, format=dd} <= 15" using string variables for both sides of the condition builder.
        2. Set the result to "First Half".
        3. Configure the action to "Set Result Only".
      2. Action 2
        1. Set the condition for the first action as "{CURR_DATE, format=dd} >= 16" using string variables for both sides of the condition builder.
        2. Set the result to "Second Half".
        3. Configure the action to "Set Result Only".
  4. Configure the Evaluation Frequency tab to run the goal hourly.
  5. Evaluate the Goal. If the date is less than or equal to the 15th of the month, the result should be "First Half". If the date is later than or equal to the 16th of the month, the result should be "Second Half".
