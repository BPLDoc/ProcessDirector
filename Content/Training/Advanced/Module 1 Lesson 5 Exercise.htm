Skill Exercise: Exporting Knowledge Views

Exercise Performance Tasks

Instructions

Complete the following tasks:

  1. Using an existing Knowledge View that returns more than one result when run, create an Excel template that uses data from more than one column of the Knowledge View.
  2. Upload the new excel template to the Content List
  3. Configure the Knowledge View to export to Excel when run, and use the new Excel Template as the output file for the Knowledge View.
  4. Run the Knowledge View to ensure that it exports properly to Excel.
