Lesson 7: Process Administration

Exercise Knowledge Quiz

Instructions

Match the Process Timeline administration command to its correct definition:

______ Cancel Activity

______ Restart Activity

______ Cancel Timeline

______ Add User(s)

______ Add Me to This Task

______ Cancel User

______ Complete User Task

______ Remove User

______ Restart User

______ Resend Email

______ Reassign User

  1. Adds additional users to the process step.
  2. Adds you to the process step as a participant.
  3. Restart this user task.
  4. Sets the user's task as complete.
  5. Resend the task notification email to the user.
  6. Cancels the user's participation in the task. Restarts this activity in the timeline. User, Wait, Custom Task, or Script Activities may all be restarted.
  7. Assign a different user to the task.
  8. Cancels the entire running timeline instance.
  9. Remove the user from the task.
  10. Cancels this activity in the timeline.
