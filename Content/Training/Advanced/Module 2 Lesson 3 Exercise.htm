Skill Exercise: Timeline Branching

Exercise Performance Task

Instructions

Complete the following tasks:

  1. Open the Process Timeline you created for the previous lesson.
  2. Add a new result to the Director Approval activity, named "Return to Manager".
  3. Add a new Branch activity after the Director Approval activity, and name it "Return to Manager".
  4. Make the Return to Manager activity dependent on the Director Approval activity.
  5. Set the Return to Manager activity to rollback to the Manager Approval activity when the result of the Director Approval activity is "Return to Manager.
  6. Test your rollback to ensure that it works.
  7. Save the Process Timeline for use in later lessons.
