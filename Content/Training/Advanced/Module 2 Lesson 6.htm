Lesson 6: Starting Processes from an Array

Lesson Contents In this Lesson, you will learn the configuration options for the Run Process Custom Task that enables you to use each row in an array to start a process or subprocess.

Run Process Custom Task

This custom task enables you to launch a new process. You can either specify the process within the custom task configuration, or have the process specified by a form field or an array of form fields. The Launch Data tab allows you to specify a form control that will specify the process to be launched. Alternatively, you can specify the process to be launched right in that tab.

Launch Data/New Form/Attachments Tabs

These tabs enable you to configure how the Process will be launched for each row in the array.

The Launch Process property allows you to use a Picker control to choose the process to launch, or to choose a Form Control that identifies the property from the container Form. If you choose to use a Form field to specify the process, then when the Run Process Custom Task is launched from a Form, the launching Form is assumed to be the container Form that contains the array. When the Custom Task is launched from a process, you should set the Container Form in the Custom Task tab of the Process Timeline activity.

The New Form property of the New Form tab is an object picker control from which you can select the new form to attach to the process. Multiple forms can be added by clicking the Add Mapping button.

Array Binding Tab

The Array Binding tab is where you specify the array to bind to the Custom Task to start a new process for each of the rows in the array.

The Array Field property is a dropdown control that contains a list of the arrays in the container Form. Process Director will launch a process for each row in the selected array.

The For each row, launch new Process only under the following condition property enables you to set the conditions for starting a process for each row in the array that meets the conditions you specify. When this option is checked, a list of fields, list of operators, and a text box appear to allow you to set a condition that must apply for each row in the array before that row initiates the process.
