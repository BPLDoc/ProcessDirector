Skill Exercise: Configuring the Sample Process Timeline

Exercise Performance Tasks

Instructions

  1. Create a new user activity named "Finance Approval".
    1. Set the Instructions to "Please review this Purchase Request, and make the appropriate disposition."
    2. Set the Participants to the Finance Approver business rule.
    3. Set the dependency to the VP Approval activity.
    4. Set the needed when condition to "(Activity Result [Manager Approval] = Approved AND Activity Result [VP Approval] <> Disapproved)".
    5. Set the activity to "Assign task to first user to accept".
  2. Create a new notification activity named "Initiator Notification".
    1. Set the dependency to the Finance Approval activity.
    2. Set the notification to be sent to the Timeline Initiator.
