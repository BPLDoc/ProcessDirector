Skill Exercise: Configuring the Sample Knowledge Views

Exercise Performance Tasks

Instructions

  1. Create a Knowledge View with the following attributes:
    1. Name the Knowledge View, "All PR Requests".
    2. Show only Form instances from the Purchase Request Form.
    3. Disable all actions except the ability to run the Form.
    4. Show only the following Form Fields in the Knowledge View:
      • Submitter
      • PR Number
      • Vendor Name
      • Submit Date
      • Subtotal
