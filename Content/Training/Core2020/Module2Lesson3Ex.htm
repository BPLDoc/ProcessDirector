Lesson 3 Exercise: Creating a Parent Activity

Practice Exercise Configure a Parent Activity

Instructions

Create a Process Timeline by following the directions below.

1. In the Student Exercise Folder, open the Change of Student Major.

2. Open the Properties dialog box for Activity 1

3. Change the name of Activity 1 to "Approval Loop".

4. Set the Activity Type to "Parent".

5. In the Looping tab, check the Cancel the remainder of the loop when condition is met property.

6. Use the Condition Builder to create a condition that specifies "Any Child Activity Result Contains 'Deny'".

7. Update the Process Timeline to save your changes.
