Course Objectives

Upon completion of this course, the student will be able to successfully:

  • Create and design a form, using the Online Form Designer.
  • Use Process Director to configure Form fields, events, and conditions for a Form definition.
  • Correctly create and configure a Process Timeline and Process Timeline activities to model a simple process.
  • Create and a configure business rule to return a desired value.
  • Create and configure a Knowledge View to return a desired set of objects that matches a desired set of conditions.
  • Explain how Forms interact with Process Timelines.
  • Define the term “dependency” as it applies to Process Timelines.
  • Describe the term “condition”.
