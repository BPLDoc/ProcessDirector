Lesson 4 Exercise: Creating a User Activity

Practice Exercise Configure a User Activity

Instructions

Configure the Change of Student Major Process Timeline by following the directions below.

  1. In the Student Exercise Folder, open the Change of Student Major Process Timeline.
  2. Create four (4) child activities for the Approval Loop Parent activity. When you are done, your Process Timeline should look like the example below:
  3. Configure Activity 1.1 as follows:
    1. Name the Activity "Faculty Advisor Approval.
    2. Set the duration to 2 business days.
    3. Set the participant to the Faculty Advisor form field.
    4. Create 2 results for the activity: Approve and Deny.
    5. Set the Order for the results so that Approve is ordered as 1, and Deny is ordered as 2.
