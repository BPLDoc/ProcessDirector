Lesson 1 Exercise: Completing the Sample Form

Practice Exercise Complete the form's design.

Instructions

Complete the sample form by following the directions below.

  1. Open the Online Form Designer by clicking on the Edit tab of the Form definition.
  2. In the "Desired Degree:" row, add a new Radio Button List control with the following properties:
    1. Name: DegreeDesired
    2. Items:

      BA

      BFA

      BM

      BS

  3. In the "Faculty Advisor Comments" area, add an Input control for the "Advisor Comments:", with the following properties:
    1. Name: FacultyAdvisorComments
    2. Columns: 40
    3. Rows: 5
  4. In the "Department Approval" area, add the following controls:
    1.  For the "Department Representative:" add a User Picker control named "DepartmentRep".
    2.  For the "Comments:", add an Input control with the following properties:
      1. Name: DepartmentRepComments
      2. Columns: 40
      3. Rows: 5
  5. In the "Dean's Office Approval" area, add the following controls:
    1.  For the "Dean's Office Representative:" add a User Picker control named "DeansOfficeRep".
    2.  For the "Comments:", add an Input control with the following properties:
      1. Name: DeansOfficeRepComments
      2. Columns: 40
      3. Rows: 5
  6. In the "Faculty Advisor Assignment" area, add a User Picker control named "NewFacultyAdvisor".
