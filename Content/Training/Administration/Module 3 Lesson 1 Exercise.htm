Skill Exercise: Administering Users

Exercise Performance Tasks

Instructions

Complete the following tasks.

  1. Navigate to User Administration.
  2. Create a new built-in user.
    1. Set the User ID to "testuser".
    2. Set the password as "password".
    3. In the User Edit screen, set the user's name to "Test User".
    4. Set the default time zone to Pacific time.
    5. Click the OK Button to save the user.
  3. Delete the User you just created.
