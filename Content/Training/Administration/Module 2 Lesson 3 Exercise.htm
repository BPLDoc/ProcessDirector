Skill Exercise: Configuring Global Knowledge Views

Exercise Performance Tasks

Instructions

  1. Create a new Process Director Global Knowledge View.
    1. Name the partition "Sample Global Kview".
    2. Enter the Description as "Sample Global Knowledge View for training."
    3. Set the icon to an appropriate icon.
    4. On the Advanced Options Tab:
      1. Allow these Actions on Objects: Clear all the check boxes in this section.
      2. Return These Object Types: Ensure that only Workflow Instances and Timeline Instances are checked.
      3. Display Action Icons For: Ensure that only View is checked.
      4. Users Will Have Permission to Create: Clear all the check boxes in this section.
    5. On the Columns Tab, set the following column values:
      1. Process Name: Process Instance name
      2. Run Time: Process Run Time
      3. Users: All Process Users
      4. Started On: Process Start Date
    6. On the Filter tab, set the Knowledge View to show only active processes.
