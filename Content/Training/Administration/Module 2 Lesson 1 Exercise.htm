Skill Exercise: Workspace Configuration

Exercise Performance Tasks

Instructions

Complete the following tasks.

  1. Create a new Process Director Workspace.
    1. Name the workspace "Training Admin".
    2. Give it a description of "Sample workspace for training administrators."
    3. Change the default workspace icon to an appropriate icon.
    4. Display both text and icon in the workspace tab.
    5. Assign the Admin group to the workspace.
  2. Set the Top Navigation Buttons as described below.
    1. Create a home page button, using an appropriate icon.
    2. Create a logoff button using an appropriate icon.
  3. Set the Home Page Windows as described below.
    1. Set the page to display two vertically stacked Portlets.
    2. The top Portlet should display the Task List.
    3. The bottom Portlet should display the "Forms I can Submit" Knowledge View.
  4. Save and close the Workspace configuration screen.
